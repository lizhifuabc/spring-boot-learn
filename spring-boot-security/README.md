## 认证（Authentication）和授权（Authorization）

认证：确认身份

授权：权限

例如使用用户名密码进行登录认证，操作具体的模块时是否有权限。



## 异常

Spring Security 中异常主要分为两大类

- AuthenticationException: 认证异常
- AccessDeniedException: 授权异常
